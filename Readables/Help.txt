Workflow - GIT BASH and VS

1. Make changes to code in index.html or style.css or js.
2. Open git bash
3. check status with ' git status '
4. stage changes with ' git add . '
5. commit changes with ' git commit -m "YADAYADYAD" '
6. git push origin main
~~~~~~~~~~~~~~~~~~~~~~
Optional:
To see history of commits use: ' git log '
